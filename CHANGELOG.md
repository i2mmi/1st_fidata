# 변경 사항 요약

## ✅ 완료된 작업

### 1. CSV 파일 정리
- 모든 CSV 파일을 `data/` 폴더로 이동
- 파일 구조 정리 완료

### 2. 업종명 기반 섹터 분류
- `sector_mapper.py` 모듈 생성
- `data/data_kospi.csv`의 실제 업종명을 사용하여 섹터 자동 분류
- 하드코딩된 섹터 매핑 제거 → 실제 업종 데이터 사용

### 3. Streamlit 앱 개선
- 업종명 기반 섹터 분류 적용
- CSV 파일 기본 선택 옵션 추가
- 기존 디자인 유지

## 🔄 섹터 분류 방식 변경

### 이전 방식
- 하드코딩된 종목명 리스트로 매칭
- 대부분의 종목이 "기타"로 분류됨

### 새로운 방식

- 업종명 키워드 기반 자동 매핑
- 더 정확한 섹터 분류

## 🚀 사용 방법

### Streamlit 앱
```bash
streamlit run app.py
```

1. 사이드바에서 데이터 소스 선택
2. "CSV 파일" 선택 시:
   - 기본 파일 사용: `data/kospi_code_list_100.csv` 등 선택
   - 파일 업로드: 직접 CSV 업로드
3. 분석 시작
